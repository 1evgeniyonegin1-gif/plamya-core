## Пост 1: B2B лидген

**Заголовок:** Как я автоматизировал поиск клиентов: 1800 бизнесов за месяц

Менеджер по продажам обзванивает 20-30 компаний в день. Находит контакты вручную, пишет КП в Word, отправляет по одному.

Я построил систему, которая делает это автоматически:
- Парсит бизнесы из 5 источников (2GIS, Google Maps, каталоги, VK, поиск)
- AI анализирует каждый бизнес и находит слабые места
- Генерирует персонализированное КП под конкретную компанию
- Рассылает через Email, WhatsApp, Telegram, VK

Результат: 1800+ бизнесов обработано. Один человек делает работу отдела из 5 менеджеров.

Стек: Python, Playwright, Claude AI, SQLite
GitHub: github.com/dlysenko-dev/plamya-core

**Заголовок:** MVP мультитенантной SaaS-платформы за 1 день. Как?

## Пост 3: AI-психолог с RAG

**Заголовок:** Написал AI-психолога с собственной базой знаний. Зачем?

Проблема: обычный ChatGPT даёт поверхностные ответы на вопросы о когнитивных искажениях. Он не помнит контекст прошлых сессий и не знает специализированную литературу.

Решение: full-stack приложение с RAG-пайплайном.

Как работает:
1. База знаний из книг по КПТ загружена в PostgreSQL с pgvector
2. Каждый запрос пользователя превращается в вектор (sentence-transformers, 384 измерения)
3. Система находит 5 самых релевантных фрагментов из базы
4. Claude получает контекст + историю сессий и даёт глубокий ответ

Frontend: React + TypeScript + Tailwind (не просто чат — полноценный интерфейс).
Telegram Mini App для мобильного доступа.

Стек: FastAPI, React, PostgreSQL, pgvector, sentence-transformers, Claude
GitHub: github.com/dlysenko-dev/ai-psychologist
